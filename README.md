# Portfolio — Folly Nelson Emmanuel

Site portfolio personnel construit avec Next.js 16, TypeScript, Tailwind CSS et next-intl.

## Stack

- **Framework** : Next.js 16 (App Router, Turbopack)
- **Language** : TypeScript
- **Styling** : Tailwind CSS 4
- **i18n** : next-intl v4 (FR/EN)
- **Analytics** : Vercel Analytics
- **Deploy** : Vercel

## Pages

- **Accueil** — Presentation et CTA
- **About** — Parcours, formation, experiences
- **Projets** — 6 projets Data/ML
- **Realisations** — Certifications
- **Dashboard** — Stats GitHub, Wakatime, Codewars
- **Contact** — Formulaire email

## Installation

```bash
git clone https://github.com/moltesse12/Portfolio-IA.git
cd Portfolio-IA
npm install
cp .env.example .env
npm run dev
```

## Variables d'environnement

Voir `.env.example` pour la liste complete.

## Deploy

Le site est deploye sur Vercel. Tout push sur `main` declenche un deploiement automatique.
